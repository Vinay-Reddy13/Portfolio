### Adding New Projects
To add a new project, copy the project card structure in the HTML:

```html
<div class="project-card">
    <div class="project-image">
        <i class="fas fa-project-icon"></i>
    </div>
    <div class="project-content">
        <h3>Project Name</h3>
        <p>Project description...</p>
        <div class="project-tech">
            <span class="tech-tag">Technology</span>
        </div>
    </div>
</div>
```

### Updating Skills
Add new skills by modifying the skills section in the HTML:

```html
<div class="skill-category">
    <h3><i class="fas fa-icon"></i> Category Name</h3>
    <div class="skill-tags">
        <span class="skill-tag">New Skill</span>
    </div>
</div>
```

### Contact Form
The contact form includes client-side validation and a notification system. To connect it to a backend service, modify the form submission handler in `script.js`.
